# entrainement medecine

Projet de création d'une plateforme en ligne permettant de réaliser des entrainements interactifs permettant d'apprendre des cours de medecine.
- application associée
- les questions sont triées selon les cours auxquelles elles font reference
- l'utilisateur peut créer des sessions d'entrainement et preciser le nombre de questions qu'il souhaite effectuer pendant sa session ainsi que leurs themes
- les questions sont corrigées selon un barème spécifique aux EDNs
- l'utilisateur peut choisir de réaliser des questions en "format flash" (avec la correction qui apparait quand l'utilisateur submits ses réponses pour une question) ou "format classique" (les corrections sont visibles une fois toutes les questions submitted)
- un compte admin qui permet d'ajouter des questions et de modifier les pages
- plusieurs utilisateurs ayant chacun leur propre compte
## pyfoundry-template

Ce projet a été instancié depuis un template.

Toutes les infos sur ce template et la façon de l'utiliser sont dans sa [documentation](https://castorfou.github.io/PyFoundry/).

# Système de trophées

La plateforme intègre un système de trophées inspiré des achievements PlayStation. Les trophées récompensent votre progression et s'affichent dans l'onglet **Trophées** de votre profil.

## Niveaux de rareté

| Couleur | Niveau | Description |
|---------|--------|-------------|
| 🟤 Bronze | Débutant | Premiers pas sur la plateforme |
| ⚪ Argent | Intermédiaire | Pratique régulière |
| 🟡 Or | Expert | Maîtrise avancée |

## Obtenir un trophée

Les trophées se débloquent automatiquement lorsque vous remplissez leurs conditions. Une notification apparaît brièvement en haut à gauche de l'écran pendant 3 secondes.

## Consulter vos trophées

Rendez-vous sur votre **page de profil** (`/profil/`) puis cliquez sur l'onglet **🏆 Trophées**.

- Les trophées obtenus s'affichent avec leur couleur (or, argent ou bronze) et la date d'obtention.
- Les trophées non obtenus s'affichent en noir avec un cadenas 🔒.
- Certains trophées sont **masqués** : leur nom et leurs conditions s'affichent en "???" tant qu'ils ne sont pas débloqués.
- Le pourcentage d'étudiants ayant débloqué chaque trophée est affiché sous chaque carte.

## Types de conditions

| Type | Description |
|------|-------------|
| Questions réalisées | Nombre total de questions auxquelles vous avez répondu |
| Bonnes réponses | Nombre de questions correctement répondues |
| Questions/réponses par matière | Idem, filtré sur un enseignement (EC) |
| Session parfaite | Obtenir 20/20 sur une session entière |
| Sessions complètes | Sessions de **10 questions ou plus** entièrement répondues |
| Erratas acceptés | Signalements de qualité validés par l'administration |
| Questions à 0 point | Questions sur lesquelles vous n'avez obtenu aucun point |
| Questions à 0 point (par matière) | Idem, filtré sur un EC |
| Connexions | Nombre total de connexions à la plateforme |
| Jours consécutifs | Plus longue série de jours de connexion consécutifs |

## Organisation par année

Les trophées sont regroupés par année d'étude dans l'affichage :

1. **Transversal** — trophées valables en P2 et D1
2. **P2** — trophées spécifiques à la deuxième année
3. **D1** — trophées spécifiques à la première année de doctorat
4. **Non classé** — trophées sans année assignée
